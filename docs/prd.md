# MiniPay Yield Copilot PRD

## Product thesis

MiniPay Yield Copilot gives MiniPay users one clear answer about what to do
with idle stablecoins: stay liquid, use a MiniPay-native earning path, or move
into a narrowly approved third-party venue on Celo.

## MVP scope

- MiniPay-first frontend with auto-connect aware shell
- stablecoin-only recommendation flow
- deterministic recommendation engine with one primary and one backup venue
- plain-language risk framing before execution
- user-approved execution plan
- active position and alerts views

## Non-goals

- autonomous fund movement
- leveraged strategies
- cross-chain routing
- long-tail protocol discovery
- generic yield aggregation

## v1 venues

- Stay liquid in MiniPay
- MiniPay Boost
- Kiln inside MiniPay
- one optional direct Celo venue when integration complexity is acceptable
